# To-Do List

Aplicação web de lista de tarefas desenvolvida com HTML, CSS e JavaScript.

## Funcionalidades

- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Adicionar tarefa pressionando Enter

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
